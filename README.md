# VantaBank
VantaBank is a native banking application developed using JavaFX. It allows users to manage their bank accounts, perform transactions, and much more, all within a modern and intuitive user interface.

## Features
With VantaBank, you can manage your bank account. It allows you to create, modify, or delete an account and perform transactions between them. The admin panel lets you manage every customer account efficiently.
